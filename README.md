#  House Prices: Advanced Regression Techniques (CatBoost)

##  Задача
Регрессия: предсказать итоговую стоимость дома на основе 79 признаков (площадь, материалы, локация, год постройки и др.). Метрика соревнования: RMSE по логарифмированным ценам.

##  Стек технологий
- **Язык**: Python 3.x
- **Обработка данных**: `pandas`, `numpy`
- **Моделирование**: `catboost` (`CatBoostRegressor`)
- **Валидация**: Hold-out 80/20 + ранняя остановка (`early_stopping_rounds=50`)

##  Пайплайн решения
1. **Очистка признаков**: 
   - Удалены константы (`nunique == 1`) и колонки с >80% пропусков
   - Удалены дубликаты колонок и явный шум (`Utilities`)
2. **Синхронизация train/test**: `align(join='inner')` гарантирует идентичный набор признаков
3. **Обработка пропусков**:
   - Числовые → медиана из обучающей выборки
   - Категориальные → приведение к `str`, замена `NaN/None` на маркер `'Missing'`
4. **Таргет-трансформация**: `np.log1p(SalePrice)` для снижения влияния выбросов и выравнивания распределения
5. **Модель**: `CatBoostRegressor` с автоматическим `Ordered Target Encoding` категорий
6. **Валидация**: оценка на hold-out части с возвратом к исходной шкале (`np.expm1`)

##  Результаты
| Метрика | Значение |
|---------|----------|
| **Validation RMSE (original scale)** | `~192 853` |
| **Лучшие гиперпараметры** | `iterations=1000, depth=6, lr=0.05` |
| **Ранняя остановка** | Сработала через `~150–200` итераций (зависит от фолда) |

>  **Примечание**: Kaggle оценивает RMSE по `log(SalePrice)`. Валидация на оригинальной шкале дана для интерпретируемости. На публичном лидерборде ожидаемый RMSE (log-scale) ~ `0.11–0.13`.
